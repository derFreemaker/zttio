# zttio
TTY interface with main focus on making it easy to get input from the terminal like keystrokes and mouse interactions.

## Third-Party
- [zigwin32](https://github.com/marlersoft/zigwin32)
- [uucode](https://github.com/jacobsandlund/uucode)
- [zigimg](https://github.com/zigimg/zigimg)
- [libvaxis](https://github.com/rockorager/libvaxis) (a lot of inspirations)
